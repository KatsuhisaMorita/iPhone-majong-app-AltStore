# 麻雀スコア記録アプリ 要件定義書 (Requirements Document)

## 1. アプリの目的
iPhone上で、麻雀（4人打ち）のスコア・成績を簡単に記録・管理するためのアプリケーション。
紙のスコアシートや計算の煩わしさを解消し、日々の成績（セッション）とプレイヤーごとの累計成績を可視化する。

## 2. 画面・UI構成 (すべて日本語表記)
1. **トップ画面 (ContentView)**
   - 「本日の対局を開始」ボタン（タップ時に参加するプレイヤー4名を選択してセッションを作成）
   - 「本日の対局を再開」リンク（既にその日に対局がある場合）
   - 「新しい対局を開始」ボタン（途中でメンツが変わった場合などのための新セッション作成）
   - 過去の対局履歴一覧
   - プレイヤー管理へのリンク
   - ルール設定へのリンク
2. **本日の対局 (DailySessionView)**
   - 参加しているプレイヤー4名を表示
   - その日のプレイ数（〜半荘）と、各プレイヤーの本日の合計スコア（浮き沈み）を表示
   - その日に打った半荘の履歴リスト（順位とスコア）
   - 「新規半荘入力」ボタン
   - 「セッション終了（チップ入力）」ボタン
3. **新規半荘入力 (GameInputView)**
   - 各プレイヤーの持ち点（素点）を入力。3人分入力すると4人目は自動計算される。
   - 4人の合計点が設定値（例: 100,000点）と一致しない場合はエラーとなり保存できない
   - ※ 完全に同点のプレイヤーが発生した場合のみ追加で「席順（起家〜北家）」を入力するUIを表示する。
4. **セッション終了時チップ入力 (ChipInputView / Alert)**
   - セッション完了時に各プレイヤーの獲得チップ枚数をまとめて入力する。
4. **プレイヤー実績 (PlayersListView)**
   - 登録済みの全プレイヤー一覧
   - 累計対局数（半荘数）、累計スコア
   - 平均順位、連対率（1位・2位になる確率）、ラス回避率（4位にならない確率）の表示
   - 新規プレイヤーの追加・削除
5. **ルール設定 (SettingsView)**
   - ウマの設定（ワンスリー 10-30、ワンツー 10-20 など。3位・4位は自動計算）
   - 飛び賞（箱下）のオン・オフ機能とボーナス/ペナルティ点数の設定
   - チップ1枚あたりのスコア（ポイント）設定

## 3. スコア計算ロジック
- **持ち点・返し点**: デフォルトは25000点持ち、30000点返し。（トップ賞（オカ）は +20点 となる計算）
- **ポイント換算**: （持ち点 - 30000点）÷ 1000 で計算。
  - **五捨六入**: 端数処理として「五捨六入」を採用。例として、+500点は切り捨て（0）、+600点は切り上げ（+1）、-500点はゼロに近づく（0）、-600点はマイナスが大きくなる（-1）。
- **ウマ**: 設定されたウマの値を1位（+）、2位（+）、3位（-）、4位（-）に加減算。
- **飛び賞（箱下）**: オプションで有効にした場合、持ち点が0点未満（マイナス）になったプレイヤーにペナルティ（例: -10点）を課し、1位のプレイヤーにボーナス（例: +10点）を付与する。
- **チップ**: セッション終了時にまとめて設定された1枚あたりのスコア（例: 2点）に基づいてポイントを加減算。
- **同点（タイ）の処理**: 最終スコアが完全に同点の場合は、入力された「席順」をもとに「起家（東）に近い方（東＞南＞西＞北）」を上位とする（上家優先ルール）。同点が発生していない場合は席順の入力は不要。

## 4. データ管理 (SwiftData)
- アプリ内にローカル保存（クラウド同期やアカウント登録は不要）。
- **Player**: 名前、累計ゲーム数、累計スコア、1~4位の獲得回数。
- **DailySession**: 対局日、参加プレイヤー4名の参照、その日に行われた半荘のリスト、およびセッション終了時に入力した最終チップ枚数を保持。
- **GameRecord**: 1回の半荘データ。タイムスタンプと、参加したプレイヤーのスコアリスト（PlayerGameScore）を保持。
- **PlayerGameScore**: 半荘における各プレイヤーの素点、順位、最終ポイント（同点時のみ席順）。
- **RuleSettings**: ウマ、飛び賞設定、チップレートなどの設定値。

## 5. デプロイメントと環境
- **ビルド環境**: GitHub ActionsによるクラウドMac環境でのCI/CD。手元のWindows PCからGitHubにコードをPushするだけでビルドが実行される。
- **配布先**: 生成された署名なしのIPAファイル (`MahjongScoreApp.ipa.zip`) をGitHub ActionsのArtifactからダウンロードし、Sideloadlyを使って実機（iPhone）にインストールする。
